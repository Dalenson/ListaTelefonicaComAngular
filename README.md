# Lista Telefônica com Angular

Utilizando um cadastro simples que grava informações em memoria para realizar a utilização do Angular.js.
O cadastro realiza:
* Inclusão
* Exclusão
* Busca

Tecnologias utilizadas:
* HTML
* CSS
* Angular JS
* Bootstrap

Utilizando metodos como:
* angular.copy
* filter
* some
* order by
* ng-model
* ng-options
* ng-repeat
* ng-show
* ng-click
* ng-disabled
* ng-required
* ng-controller
